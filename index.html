What is TeaBo?

TeaBo (Acronym for "Team Board) is an online collaboration platform for collecting ideas and brainstorming. This is a project of students of the Beuth Hochschule f√ºr Technik Berlin, University of Applied Sciences and still under development.

Features

With TeaBo you are able to work with many people on an imaginary, endless whiteboard to create, collect and share your ideas. Everythink happens in realtime as it would be a real whiteboard.You can create notes, images, attachments or videos, move them arround to arrange its position.

Installation

To run TeaBo some requirements are needed. Please install NodeJS and MongoDB before install TeaBo.

Get Code and Install

$ cd TARGET_DIR
$ git clone https://github.com/Qyotta/TeaBo.git
$ cd Teabo
$ sh install.sh
To run it, start mongodb and run the app with node:

$ node app.js
